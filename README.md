# 🌟 AURÉLIA – AI Fashion Stylist

An AI-powered desktop application that analyzes facial features using computer vision and suggests personalized hairstyles.

---

## 🚀 Features

- 📸 Real-time camera capture using OpenCV
- 🤖 AI-based hairstyle recommendation (Gemini API)
- 🔐 User authentication system (SQLite)
- 🔊 Voice output using text-to-speech
- 💻 Clean UI built with KivyMD

---

## 🛠️ Tech Stack

- Python
- OpenCV
- KivyMD
- SQLite
- Google Gemini API
- pyttsx3 (Text-to-Speech)
